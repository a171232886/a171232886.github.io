torch.tensorзҡ„еҲӣе»әдёҺеұһжҖ§

еҸ‘иЎЁдәҺ 2021-03-04 жӣҙж–°дәҺ 2022-02-14 еҲҶзұ»дәҺ з¬”и®° пјҢ PythonпјҢNumpyе’ҢPytorch йҳ…иҜ»ж¬Ўж•°пјҡ

д»Јз ҒеҶҷзқҖеҶҷзқҖпјҢеҸ‘зҺ°иҰҒеӯҰд№ дёҖдёӢеҹәзЎҖзҡ„зҹҘиҜҶгҖӮиҝҷйҮҢеҸӘдёҫжңҖз®ҖеҚ•зҡ„дҫӢеӯҗпјҢд»ҘдҫҝиҜҙжҳҺеҗ„еұһжҖ§зҡ„еҠҹиғҪгҖӮ

дёҖгҖҒtensorзҡ„еҲӣе»ә

жҢүе®ҳж–№зҡ„ж•ҷзЁӢ

torch.tensor(data, dtype=None, device=None, requires_grad=False, pin_memory=False) вҶ’ Tensor

жіЁпјҡ

  1. dataе°ұжҳҜж•°жҚ®пјҢиӢҘиҫ“е…Ҙ1пјҢеҲӣе»әж ҮйҮҸпјҢиӢҘиҫ“е…Ҙ[1], еҲӣе»әеҗ‘йҮҸ

  2. dtypeпјҡиҫ“е…Ҙж•°жҚ®иӢҘжҳҜж•ҙж•°пјҢdtypeй»ҳи®Өдёәtorch.int64;иҫ“е…Ҙе°Ҹж•°пјҢdtypeй»ҳи®Өtorch.floatпјҲtorch.float32е’Ңtorch.floatзӣёеҗҢпјүгҖӮеҸҜд»ҘжҢҮе®ҡж•°жҚ®зұ»еһӢпјҢжҢүз…§torch.dtype

  3. deviceпјҢй»ҳи®ӨcpuпјҢеҸҜд»ҘеҶҷвҖҳcpuвҖҷпјҢвҖҳcudaвҖҷ(й»ҳи®Ө0еҚЎ)пјҢвҖҳcuda:0вҖҷгҖӮжҢүз…§torch.device

  4. requires_grad, иҝҷдёӘtensorжҳҜеҗҰиҰҒжұӮжўҜеәҰпјҢй»ҳи®ӨFalse

дәҢгҖҒtensorзҡ„еұһжҖ§

еҲӣе»әе®Ңд№ӢеҗҺпјҢз»“жһңи°ғиҜ•дёҖзңӢпјҡвҖңеҡҜпјҢиҝҷйғҪд»Җд№ҲзҺ©ж„ҸпјҒвҖқгҖӮ

е…¶е®һиҝҷйҮҢйқўеҘҪеӨҡйғҪжҳҜ autogradйҮҚзҡ„дёңиҘҝпјҢжҢ‘еҮ дёӘйҮҚиҰҒзҡ„пјҲз”ЁиҝҮзҡ„пјүз ”з©¶дәҶдёҖдёӢгҖӮеӣ дёәж¶үеҸҠдёҖдәӣе…¶д»–йҮҚиҰҒжҰӮеҝөпјҢжүҖд»ҘиҝҷдёҖйғЁеҲҶдјҡеӨҡдёҖдәӣгҖӮ

пјҲдёҖпјүis_leafпјҢgradпјҢgrad_fn

еҒҮи®ҫжңҖз®ҖеҚ•зҡ„жғ…еҶөпјҢ \[ y=wx \\ z=y+b \]

x = torch.tensor([2.,1.])
w = torch.tensor([1.0,1.0], requires_grad=True)
b = torch.tensor(2.0, requires_grad=True)
y = w*x
z = y + b
жһ„жҲҗдёҖеј еүҚйЎ№и®Ўз®—еӣҫгҖӮ

еӣҫзүҮеј•з”ЁжқҘжәҗ

  1. е…¶дёӯwпјҢxпјҢbз§°дёәleaf nodeпјҢжүҖд»ҘиҝҷеҮ дёӘtensorзҡ„is_leaf=True,иҖҢzз§°дёәж №иҠӮзӮ№гҖӮ

  2. zе’Ңyеӣ дёәжҳҜеңЁw,bзҡ„еҹәзЎҖдёҠиҝӣиЎҢи®Ўз®—пјҢд»–д»¬зҡ„requires_gradиҮӘеҠЁдёәTrue

иҝӣиЎҢеҸҚеҗ‘дј ж’ӯ

backward()еҸӘиғҪеҜ№ж ҮйҮҸиҝӣиЎҢ

z.sum().backward()

  1. жҲ‘д»¬еҸҜд»Ҙиҫ“еҮәйӮЈдәӣis_leaf=TrueпјҢrequires_grad=Trueзҡ„жўҜеәҰдҝЎжҒҜ,жҜ”еҰӮwзҡ„жўҜеәҰгҖӮ 

    print(w.grad)
    иҫ“еҮәпјҡtensor(2.) пјҢжіЁж„ҸпјҢиҝҷжҳҜzеҜ№xзҡ„жўҜеәҰгҖӮ

  2. йқһleaf nodeзҡ„zе’ҢyпјҢgrad_fnдёҚдёәNoneпјҢgrad_fnиЎЁзӨәжұӮеҜјеҮҪж•°пјҢжҜ”еҰӮz.grad_fnдёә<AddBackward0 object at 0x7fb6b8991290>

  3. еҰӮжһңеҸӘиҝҗиЎҢz.backward()пјҢеҸӘиғҪзңӢдёәleaf nodeзҡ„gradпјҢиҖҢдёҚиғҪзңӢйқһleaf nodeзҡ„gradгҖӮgrad_fnеҸӘжңүйқһleaf nodeзҡ„иҠӮзӮ№жүҚдёҚдёәNoneгҖӮ

йӮЈеҰӮжһңиҰҒжҳҜжғізңӢzеҜ№yжұӮзҡ„жўҜеәҰе‘ў? еҸҜд»ҘеҰӮдёӢж“ҚдҪң

print(torch.autograd.grad(z,y))

пјҲдәҢпјүshapeе’Ңndim

еҸӘжңүдёҖзӮ№йңҖиҰҒжіЁж„Ҹпјҡ иӢҘжҳҜж ҮйҮҸпјҢ shapeдёәtorch.Size([])пјҢndimдёә0

иӢҘжҳҜеҗ‘йҮҸпјҲзҹ©йҳөпјүпјҢshapeе°ұжҳҜз»ҙеәҰжҜ”еҰӮtorch.Size([1, 3, 127, 127])пјҢndimжҳҜз»ҙеәҰж•°пјҢжҜ”еҰӮ4

пјҲдёүпјүTensor.data, Tensor.detach()е’ҢTensor.detach_()

з»“и®ә

  1. дёүиҖ…зҡ„дҪңз”Ёзұ»дјјпјҢйғҪеёёз”ЁдәҺжўҜеәҰжҲӘж–ӯгҖӮ

  2. еҸӘиҰҒдёҚйңҖиҰҒеҜ№tensorеҺҹеҶ…е®№жӣҙж”№пјҢдёүдёӘйғҪеҸҜд»ҘгҖӮдҪҶжҺЁиҚҗdetach()

  • дёӢйқўиҝҷз§ҚпјҢе°ұдёҚжҳҜеҜ№еҺҹеҶ…е®№жӣҙж”№ 
    y1=y.detach()
y1=y1+1
  • дёӢйқўиҝҷз§ҚеҶҷжі•пјҢжүҚдјҡеҜ№еҺҹеҶ…е®№дә§з”ҹеҪұе“Қ 
    y1=y.detach()
y1[:]=y1[:]+1

е…·дҪ“жқҘиҜҙпјҢдёүиҖ…зҡ„еҢәеҲ«жҳҜ

еҸӮиҖғдәҺpytorch .detach() .detach_() е’Ң .dataз”ЁдәҺеҲҮж–ӯеҸҚеҗ‘дј ж’ӯ

  1. detach()
  • иҝ”еӣһдёҖдёӘж–°зҡ„VariableпјҢд»ҺеҪ“еүҚи®Ўз®—еӣҫдёӯеҲҶзҰ»дёӢжқҘзҡ„пјҢдҪҶжҳҜд»ҚжҢҮеҗ‘еҺҹеҸҳйҮҸзҡ„еӯҳж”ҫдҪҚзҪ®,дёҚеҗҢд№ӢеӨ„еҸӘжҳҜrequires_gradдёәfalseпјҢеҫ—еҲ°зҡ„иҝҷдёӘVariableж°ёиҝңдёҚйңҖиҰҒи®Ўз®—е…¶жўҜеәҰпјҢдёҚе…·жңүgradгҖӮ
  • еҪ“дҪҝз”Ёdetach()пјҢдҪҶжҳҜжІЎжңүеҜ№еҺҹеҶ…е®№иҝӣиЎҢжӣҙж”№ж—¶пјҢе№¶дёҚдјҡеҪұе“Қbackward()
  • еҪ“дҪҝз”Ёdetach()пјҢеҜ№еҺҹеҶ…е®№иҝӣиЎҢжӣҙж”№ж—¶пјҢbackward()еҲҷдјҡжҠҘй”ҷ
  1. detach_()
  • зӣёеҪ“дәҺеҸҳйҮҸд№Ӣй—ҙзҡ„е…ізі»жң¬жқҘжҳҜx -> m -> y,иҝҷйҮҢзҡ„еҸ¶еӯҗvariableжҳҜxпјҢдҪҶжҳҜиҝҷдёӘж—¶еҖҷеҜ№mиҝӣиЎҢдәҶ.detach_()ж“ҚдҪң,е…¶е®һе°ұжҳҜиҝӣиЎҢдәҶдёӨдёӘж“ҚдҪңпјҡ
  • е°Ҷmзҡ„grad_fnзҡ„еҖји®ҫзҪ®дёәNone,иҝҷж ·mе°ұдёҚдјҡеҶҚдёҺеүҚдёҖдёӘиҠӮзӮ№xе…іиҒ”пјҢиҝҷйҮҢзҡ„е…ізі»е°ұдјҡеҸҳжҲҗx, m -> y,жӯӨж—¶зҡ„mе°ұеҸҳжҲҗдәҶеҸ¶еӯҗз»“зӮ№
  • з„¶еҗҺдјҡе°Ҷmзҡ„requires_gradи®ҫзҪ®дёәFalseпјҢиҝҷж ·еҜ№yиҝӣиЎҢbackward()ж—¶е°ұдёҚдјҡжұӮmзҡ„жўҜеәҰ
  1. data дёҺdetachзҡ„дёҚеҗҢд№ӢеӨ„пјҡеҪ“дҪҝз”Ё.data пјҢеҜ№еҺҹеҶ…е®№иҝӣиЎҢжӣҙж”№ж—¶пјҢbackward()е°Ҫз®ЎдёҚдјҡжҠҘй”ҷдҪҶиҫ“еҮәзҡ„ж•°жҚ®еҚҙжҳҜй”ҷиҜҜзҡ„гҖӮ

еҰӮдҪ•жўҜеәҰжҲӘж–ӯ

#еүҚжңҹеӨ„зҗҶиҝҮзЁӢ
x = torch.tensor([2.,1.])
w = torch.tensor([1.0,1.0], requires_grad=True)
b = torch.tensor(2.0, requires_grad=True)
y = w*x
z = y + b

#еҗҺжңҹдҪҝз”ЁиҝҮзЁӢпјҢеҸӘдҪҝз”ЁеҲ°yдҪңдёәиҫ“е…ҘпјҢиҖҢдёҚеҶҚйңҖиҰҒеҜ№еүҚжңҹеӨ„зҗҶзҡ„жўҜеәҰ
c = y.detach()
d = torch.tensor([1.,2.],requires_grad=True)
c2 = c*d
c2.sum().backward()
print(x.grad)
print(d.grad)
